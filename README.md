# 🔐 Password Manager

A full-stack Password Manager with secure authentication and encryption, allowing users to safely store, manage, and retrieve their credentials. The system ensures data privacy using encryption techniques and structured access control.

---

## 🚀 Features

- 🔐 **Authentication & Authorization**
  - Secure login/signup system
  - JWT-based authentication

- 📧 **OTP Verification (Email)**
  - Email-based OTP verification using Nodemailer
  - Secure account verification and password reset

- 🔑 **Password Management**
  - Add, edit, and delete credentials
  - Store sensitive data securely

- 🔒 **Security**
  - Password hashing (bcrypt)
  - Data encryption (AES or similar)

- 🔍 **Search & Organization**
  - Easily find stored credentials

---

## 🧑‍💻 Tech Stack

- **Frontend:** React.js 
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB
- **Security:** JWT, bcrypt, Encryption  
- **Other:** Nodemailer (OTP email service)
